# Enterprise Work Management System

A full-stack **Enterprise Work Management System** built using **React.js, Node.js, Express.js, and MongoDB**. This application enables teams and organizations to efficiently manage projects, assign tasks, collaborate with team members, and monitor project progress through a centralized dashboard.

The system provides secure authentication, project and task management, team collaboration, reporting, and search functionality, making it an ideal solution for small and medium-sized teams.

---

## 📖 Overview

The Enterprise Work Management System helps organizations streamline their workflow by providing a centralized platform to manage projects, tasks, and team members.

This application is designed for:

- Admin
- Project Manager
- Team Members

# 🚀 Getting Started

## Clone Repository

```bash

```

---

## Backend Setup

```bash
cd backend

npm install

npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm start
```
